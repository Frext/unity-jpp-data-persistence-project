# Simple Breakout

[![Unity Version](https://img.shields.io/badge/Unity-2021.3.9f1-blue.svg?style=flat&logo=unity)](https://unity.com/)
[![Language](https://img.shields.io/badge/Language-C%23-green.svg?style=flat&logo=c-sharp)](https://learn.microsoft.com/en-us/dotnet/csharp/)
[![Platform](https://img.shields.io/badge/Platform-Windows-orange?logo=windows)](https://www.microsoft.com/windows/)

This is a Unity project demonstrating data persistence and save/load functionality for game state management.

---

## 🚀 Quick Start

Want to skip the code and just play?

1. **[Download the Repository](https://github.com/Frext/unity-jpp-data-persistence-project/archive/refs/heads/main.zip)**
2. Extract the ZIP file.
3. Open the **`BuildWindows`** folder and launch **`SimpleBreakout.exe`**.

---

## 💾 Data Persistence Features

* **Save/Load System:** Persist player progress between sessions.
* **Leaderboard:** Track and display high scores locally.
* **Data Serialization:** Efficient storage and retrieval of game data using JSON.
